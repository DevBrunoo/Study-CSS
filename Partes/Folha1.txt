Para criar arquivos direto do terminal, utilizando comando de linha basta colocar touch mais o nome do arquivo

Quando voce considerar o CSS para um site, primeiro pense apenas em uma folha de estilo externa, e nao na marcacao
 o que significa que as alteraçoes em todo o site sao um trabalho de uma folha so. 
Uma vez que o navegador acessar a folha de estilo ela sera colocada em cache e nao sera, necessario o seu download.
O resultado sao paginas que sao rederizadas mais rapidos mas tambem a economia maior em largura de banda. 

Pergunte a um desenvolvedor CSS os beneficios dessa tecnologia, e ele falara um monte com isso pergunte como ele organiza 
a sua folha de estilo e tera uma resposta vaga. 
Querendo ou nao a folha comeca pequena e termina inflada, quanto mais regras tiverem mais ela sera inflada
e importante manter as coisas bem organizadas, ver multiplas folha para uma plataforma pode se torna excesso de engenharia que 
acarreta em mal gerenciamento. 
a melhor abordagem e usar pastas para separar os arquivos. Nomei cada arquivo para cada arquivo html isso ajudara no gereciamento 
exemplo abaixo.

1) Crie uma pasta de alto nivel chamada CSS
2) Dentro dela crie uma plataforma que sera fornecida
3) Dentro de cada subpasta, coloque arquivos CSS 

Quando o projeto de web desenvolvimento for muito grande, uma bela maneira e dividir a folha de estilo em blogos gerenciaveis.

Selector e o que definem o estilo no css, o exemplo a seguir usa tag de paragrafo como seletor

p { 
    color: #F00; Aqui estamos definindo que tudo que estiver entre a tag <p>Sera vermelho</p>
}

E de extrema importancia deixar comentarios, porem de forma que o design da folha de estilo nao sofra com isso. Os comentarios começam com 
/* começam e terminal com essas barras e asteristico */

Muitos design preferem usar folha de estilo comentando encima de linhas tracejadas


/* Estilo padrão para paragrafos
-------------------------------------- */
p{
    color: #F00;
    font-size: 12px;
}

Agora com flag comentarios 

/* =h1 Deixa todos os cabeçalhos de topo cinza com 16px de altura
------------------------------------------------------------------*/
h1 {
    color: #333;
    font-size: 16px;
}

Adotar esses metodos e essenciais, bons desenvolvedores sabem de sua 
importancia, pois quanto mais longa a folha de estilo ficar mais complexa ela se torna
